2. Kali Linux
3. Metasploitable
4. Medusa
5. Nmap

Utilizamos a ISO "Metasploitable" que é uma isso disponibilizada pela Rapid7, que já vem com algumas vulnerabilidades abertas para fins acadêmicos e testes de laboratório.É a ISO "Kali Linux" desenvolvido pela comunidad OffSec que ja vem pre instalado algumas ferramentas para testes em ambientes controlados.

Neste desafio, foi proposto pela instrutora que façamos primeiramente um Brute force, para termos acesso a uma aplicação ou servidor.
